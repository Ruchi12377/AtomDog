StackChan Minimal Web Installer

M5Stack AtomS3R / ESP32-S3 向けの StackChan Minimal ブラウザ書き込みページです。

対応ブラウザ

Chrome または Microsoft Edge で開いてください。Firefox / Safari では動作しません。

インストール手順

  1. Chrome または Edge でこのページを開きます。
  2. AtomS3R / ESP32-S3 を USB ケーブルで PC に接続します。
  3. 下の Install ボタンを押します。
  4. 表示されたシリアルポートを選びます。
  5. 初回インストールでは、必要に応じて erase を選びます。
  6. 書き込み完了後、デバイスを再起動します。

うまく書き込めない場合

  • USB ケーブルが「充電専用」ではなく「通信対応」か確認してください。
  • シリアルモニタ、PlatformIO、Arduino IDE など、同じ COM ポートを使っているソフトを閉じてください。
  • ポートが出ない場合は、USB ドライバまたはデバイスマネージャを確認してください。
  • 書き込みモードに入らない場合は、BOOT ボタンを押しながら接続、または BOOT を押しながら再試行してください。

注意

このファームウェアには AI モデルや AI サーバー本体は含まれていません。 Whisper.cpp、llama.cpp / LM Studio / Ollama、piper-plus などのローカル AI サーバーは別途起動してください。
